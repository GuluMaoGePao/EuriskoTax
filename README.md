# 个人所得税计算小程序

本小程序根据2026年个人所得税计算规则手册设计，支持正向计税和反向倒算，生成标准化年度个税预算表，帮助个人和财务人员快速准确核算个人所得税。

## 功能特点

### 1. 多种计税模式
- **综合所得计税**：输入收入和扣除项，计算年度应纳税额
- **经营所得计税**：适用于个体工商户、个人独资企业等经营所得
- **分类所得计税**：利息、股息、红利等分类所得
- **反向倒算**：输入税额，反推税前收入

### 2. 详细的收入和扣除项
- **收入明细**：工资薪金所得、劳务报酬所得、稿酬所得、特许权使用费所得、年终奖
- **扣除项明细**：
  - 基本减除费用
  - 专项扣除（社保、公积金）
  - 专项附加扣除（子女教育、赡养老人、住房租金/贷款利息、继续教育、大病医疗）
  - 其他扣除（个人养老金、商业健康保险等）

### 3. 生成标准化预算表
- 月度个税预算表
- 分类所得计税表
- 经营所得年度预算表
- 反向倒算预算表

### 4. 税收优化建议
- 基于用户输入的智能分析
- 提供个性化的税务优化建议
- 帮助用户合理规划税务

### 5. 数据可视化
- 月度个税图表
- 税率分布饼图
- 收入构成分析图表

### 6. 导出功能
- PDF导出
- 打印功能

## 技术栈

- HTML5
- Tailwind CSS v3
- JavaScript
- Chart.js
- jsPDF (PDF导出)

## 使用方法

1. **选择计税模式**：根据个人情况选择适合的计税模式
2. **填写基本参数**：如工作月数、已预缴税额等
3. **输入收入明细**：填写各项收入信息
4. **填写扣除项明细**：根据实际情况填写各项扣除信息
5. **查看计算结果**：系统自动计算并生成预算表
6. **查看优化建议**：根据个人情况获取税务优化建议
7. **导出或打印**：可将预算表导出为PDF或直接打印

## 注意事项

- 本小程序仅供参考，实际纳税以税务部门核算为准
- 专项附加扣除需在个人所得税APP中据实填报，留存相关佐证材料备查
- 综合所得年度汇算清缴时间为次年3月1日至6月30日
- 经营所得年度汇算清缴时间为次年1月1日至3月31日

## 开发说明

### 项目结构
- `index.html`：主页面
- `js/`：JavaScript文件
  - `tax-calculator.js`：核心计算逻辑
  - `utils.js`：工具函数（表格更新、图表更新等）
  - `app.js`：应用逻辑
  - `helper-functions.js`：辅助函数
- `css/`：样式文件

### 本地运行

1. 克隆项目到本地
2. 使用本地服务器打开 `index.html` 文件
   - 可以使用 Python：`python -m http.server 8000`
   - 可以使用 Node.js：`npx http-server -p 8000`
3. 在浏览器中访问 `http://localhost:8000`
